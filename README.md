# Blessy Tech Academy

Parcours d'apprentissage de développement web par Jean Raymond BELONY, 
fondateur de Blessy Tech Academy et Blessy Connect.

## Structure du projet

- `apprentissage/` : exercices et mini-projets Python (Phase 1 - Fondamentaux)
- `blessy-school-manager/` : système de gestion scolaire orienté objet (en cours)

## Compétences développées

- Python (variables, conditions, boucles, fonctions, POO, gestion d'erreurs)
- Manipulation de fichiers (JSON, CSV)
- Architecture modulaire (multi-fichiers)
- Git et GitHub (branches, merge, résolution de conflits)

## À venir

- HTML, CSS, JavaScript
- Django (framework web)
- Bases de données (SQLite, PostgreSQL)
- Intelligence artificielle appliquée à l'éducation
